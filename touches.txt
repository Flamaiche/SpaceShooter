#MENU PRINCIPAL :

ENTRÉE : Valider
Flèche haut : Monter dans le menu 
Flèche bas : Descendre dans le menu 
ÉCHAP : Quitter le jeu
Bouton gauche souris : (quand elle est libre) selection et validation du menu


#EN PARTIE :

##DÉPLACEMENT
Z : Avancer 
S : Reculer 
Q : Aller à gauche 
D : Aller à droite 
ESPACE : Monter
SHIFT : Descendre

##ROTATION DE LA CAMERA
Flèche gauche : Tourner la caméra à gauche 
Flèche droite : Tourner la caméra à droite 
Flèche haut : Regarder vers le haut 
Flèche bas : Regarder vers le bas 
Déplacement de la souris : Tourner la caméra

CTRL + Flèche gauche : Tourner 2 fois moins vite la caméra à gauche
CTRL + Flèche droite : Tourner 2 fois moins vite la caméra à droite
CTRL + Flèche haut : Regarder 2 fois moins vite vers le haut
CTRL + Flèche bas : Regarder 2 fois moins vite vers le bas

##ROLL DE LA CAMÉRA (ALT + A/E/R)
ALT + A : Faire tourner la caméra à gauche
ALT + E : Faire tourner la caméra à droite
ALT + R : Réinitialiser le roll
ALT + L : Dé/Bloquer le roll (et se reset)

##MODE CAMÉRA
F5 : Passer en mode 1er personne / 3e personne

##TIR
² / Bouton gauche souris : Tirer (cooldown 0,3 s)

##GESTION DU JEU
ÉCHAP : Mettre en pause

##DEBUG
U : Menu de debug

#MENU PAUSE :

ENTRÉE : Valider
Flèche haut : Monter dans le menu 
Flèche bas : Descendre dans le menu 
ÉCHAP : Continuer le jeu
Bouton gauche souris : (quand elle est libre) selection et validation du menu


#AUTRE :

##CURSEUR
CAPS LOCK : Activer/désactiver le curseur (verrouillage/déverrouillage de la souris)
